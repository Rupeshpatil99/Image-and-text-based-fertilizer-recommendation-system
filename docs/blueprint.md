# **App Name**: FertilAIze

## Core Features:

- Image Upload & Text Input: Allow users to upload images of their crops and provide relevant text details about field conditions and prior inputs.
- AI-Powered Recommendation: Use generative AI tool to provide fertilizer recommendations based on image analysis and inputted text, including which type and quantity to use.
- Top Product Prediction: Show top fertilizer product predictions with brief explanations on why they're suitable.
- Testimonials Section: Showcase farmer testimonials (image or text-based) that describes how to successfully using fertilizer.
- Contact Form: Integrate a contact form where users can submit inquiries and request further consultation. Submissions should be sent via email to stakeholders.
- Mobile Responsiveness: Ensure the website is fully responsive and accessible on various mobile devices, maintaining a seamless user experience.

## Style Guidelines:

- Primary color: Soft green (#A7D1AB) evoking nature and growth.
- Background color: Very light desaturated green (#F4F8F2), almost white.
- Accent color: Light earthy brown (#C19A6B) that is analogous to the green, for call-to-actions and highlighting key information.
- Body and headline font: 'PT Sans', sans-serif.
- Use simple, visually clear icons to represent different types of data or functions, keeping the overall style clean and modern.
- Use a clean and modern layout with clear sections for the headline, features, testimonials, and call-to-action. Ensure a logical flow and easy navigation.
- Subtle transitions and animations (e.g., fade-in effects) can be used to enhance the user experience without being distracting.
